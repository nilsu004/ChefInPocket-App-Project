# ChefInPocket-App-Project
CS 310 Term Project

# Project Description
ChefInPocket is a smart mobile cooking assistant that helps users discover meals by cuisine and find recipes using ingredients they already have in their kitchen. The application provides an inventory first filtering system to reduce food waste and save time for users. It features category based discovery and dynamic serving adjustments to make cooking more accessible.
